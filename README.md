# CoverHomework
### Introduction
四川大学课程中心不提供作业的再次提交，一旦发现自己提交的内容出错，只能联系助教或者老师退回自己的作业，极度麻烦  
本程序就是为了解决这个问题，但是值得注意的是，本程序提供以附件方式进行覆盖.<b>本程序可以实现linux下提交作业，前提是知道作业的网站。</b>
